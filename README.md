# 分形图形生成器

一个使用 p5.js 库开发的交互式分形图形生成器，支持多种经典分形算法和实时参数调整。

## 项目功能

- **多种分形算法**：支持科赫曲线、谢尔宾斯基三角形和曼德博集合三种经典分形
- **实时参数调整**：可调整迭代深度、颜色方案和缩放级别
- **交互式体验**：通过控制面板实时预览分形效果
- **曼德博集合缩放**：支持鼠标点击进行局部缩放探索
- **响应式设计**：适配不同屏幕尺寸

## 技术栈

- **前端框架**：HTML5, CSS3, JavaScript
- **图形库**：p5.js
- **开发工具**：Visual Studio Code

## 安装和使用

### 方法一：直接打开

1. 克隆或下载本仓库到本地
2. 直接在浏览器中打开 `index.html` 文件

### 方法二：使用本地服务器

1. 克隆仓库：
   ```bash
   git clone https://github.com/yourusername/fractal-generator.git
   cd fractal-generator
   ```

2. 启动本地服务器（推荐使用 VS Code 的 Live Server 插件或 Python 服务器）：
   ```bash
   # 使用 Python 3
   python -m http.server 8000
   
   # 或使用 Python 2
   python -m SimpleHTTPServer 8000
   ```

3. 在浏览器中访问 `http://localhost:8000`

## 分形算法原理

### 1. 科赫曲线（Koch Curve）

**原理**：通过递归地将线段替换为特定的折线模式生成。

**生成步骤**：
1. 从一条直线段开始
2. 将线段三等分
3. 以中间线段为底边，构建一个等边三角形
4. 移除底边
5. 对每个新生成的线段重复上述步骤

**特点**：具有无限长度但有限面积，展现出自相似性。

### 2. 谢尔宾斯基三角形（Sierpinski Triangle）

**原理**：通过递归移除三角形中心部分生成。

**生成步骤**：
1. 从一个大三角形开始
2. 连接三边中点，将三角形分成四个小三角形
3. 移除中心的小三角形
4. 对剩余的三个小三角形重复上述步骤

**特点**：面积趋近于零，周长趋近于无限，展现出严格的自相似性。

### 3. 曼德博集合（Mandelbrot Set）

**原理**：通过复数迭代计算生成的复杂分形图案。

**生成步骤**：
1. 对于复平面上的每个点 \( c \)
2. 从 \( z = 0 \) 开始迭代 \( z = z^2 + c \)
3. 判断迭代结果是否发散（模长超过 2）
4. 根据发散速度分配颜色

**特点**：边界极其复杂，具有无限细节，是最著名的分形图案之一。

## 交互功能

### 控制面板

- **分形类型**：选择要生成的分形算法
- **迭代深度**：控制分形的复杂度（1-8级）
- **颜色方案**：选择不同的颜色风格（彩虹、灰度、蓝色系、绿色系）
- **缩放级别**：调整分形的显示大小
- **重置视图**：恢复默认参数设置

### 曼德博集合特殊操作

- **鼠标点击**：在曼德博集合视图中点击任意位置，将以该点为中心进行放大
- **多次点击**：可以连续点击进行深度缩放，探索分形的无限细节

## 核心代码说明

### 分形绘制函数

- `drawKochCurve()`：绘制科赫曲线
- `drawSierpinskiTriangle()`：绘制谢尔宾斯基三角形
- `drawMandelbrotSet()`：绘制曼德博集合

### 交互控制

- `initControls()`：初始化控制面板事件监听器
- `resetMandelbrot()`：重置曼德博集合视图参数

### 颜色管理

- `setColor()`：设置分形基本颜色
- `setMandelbrotColor()`：设置曼德博集合颜色

## 性能优化

- **递归深度控制**：限制最大迭代深度以避免性能问题
- **按需重绘**：仅在参数变化时重新绘制分形
- **曼德博集合优化**：使用数学判断提前终止发散点的计算
